# simple-MERN-CRUD-app
A simple MERN CRUD app . 

# Tech stack used 
- ExpressJs
- ReactJs
- MongoDB
- Docker

# Run Locally
```
git clone https://github.com/mdnurahmed/simple-MERN-CRUD-app
cd simple-MERN-CRUD-app
docker-compose up --build
```
Then go to localhost:3000 on browser .
